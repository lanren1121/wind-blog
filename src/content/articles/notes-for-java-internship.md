# Java 后端实习学习路线备忘

如果目标是 Java 后端日常实习，可以先把基础能力打扎实。

## 技术主线

- Java 基础：集合、并发、JVM 基础
- Spring Boot：REST API、参数校验、异常处理
- MySQL：索引、事务、慢查询分析
- Redis：缓存、过期策略、简单分布式锁
- 项目表达：业务背景、技术选型、难点和收益

## 面试表达

不要只说“我会用”，要说清楚：为什么用、怎么用、遇到什么问题、最后如何解决。
